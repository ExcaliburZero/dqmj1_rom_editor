# DQMJ1 ROM Editor
An unofficial modding tool / ROM editor for Dragon Quest Monsters Joker 1.

![Screenshot of the encounter editing screen](img/encounters.png)

## Keyboard shortcuts
* `Ctrl + s` / `Cmd + s` - Save mod
* `Ctrl + e` / `Cmd + e` - Export patched ROM

## Development
```bash
npm install
npm run tauri dev
```
