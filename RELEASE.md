## Instructions
* Update the screenshot in the readme
* Choose a new `vX.Y.Z` version number that follows [semantic versioning](https://semver.org/).
* Update the version number in the following locations:
    * `package.json`
    * `dqmj1_rom_util/Cargo.toml`
    * `src-tauri/Cargo.toml`
    * `src-tauri/tauri.conf.json`
    * `CHANGELOG.md`
* Make a commit with a message like: `Update version to vX.Y.Z`
* Create an annotated tag
```bash
export VERSION=vX.Y.Z

git tag -a $Version -m "$VERSION"
git push origin $VERSION
```
* Push the new commit
```bash
git push origin main
```
* Wait for the GitHub Actions build of `main` branch to finish
* Create a new release on GitHub
    * Use the tag you created
    * Set the release title to be `vX.Y.Z`
    * Copy paste the relevant lines from `CHANGELOG.md`
    * Download then upload to the release the following generated by GitHub Actions
        * Windows MSI installer
        * Linux AppImage
        * Linux deb
        * Linux rpm
* Create a new release on GameBanana
    * Edit the tool's page
    * Update the description
    * In the Media tab
        * Update the screenshots
        * Update the files
            * Create a zip with the name "dqmj1_rom_editor_$VERSION.zip"
            * Put all the installer/binary files in it (both Windows and Linux)
            * Copy the `README.md` and `CHANGELOG.md` into it
    * Add an update with the contents of the changelog
